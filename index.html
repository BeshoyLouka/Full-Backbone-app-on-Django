Get your app Zincified

This is a full stack boilerplate to get your app started with Django and backbone and even more.

before doing anything, make sure you have "Grunt ~0.4.0" & "Bower" installed.


$ clone the repo
$ run `pip install -r requirements.txt` to install all requirments
$ run `npm install` in the main directory where the package.json is.
$ run `python manage.py syncdb` to sync the database
$ run `python manage.py runserver` to fire up the Django server

Work Flow

The Idea is to have a aource directory (which is /app is this case) that you build all the js, css and static stuff inside, and then run the command 'grunt build' to get your 'static' dir that django relies on.

CoffeeScript, RequireJS for your Backbone

You can do all the coffeescript work in your 'app/js' directory and simply run "grunt coffee" which will compile your coffeescript into js files (thanks to the grunt-coffeescript plugin) and then you can run the magical "grunt build" command to get your main.js file inside "static/js" and get a nice "amd" support thanks to the awesomeness of require.js.
